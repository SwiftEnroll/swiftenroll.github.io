# Product & Mission

## What This Software Is

This is the **public marketing website** for SwiftEnroll (swiftenroll.github.io). It is NOT the SaaS application itself.

## Who It Serves

### Parents
Searching for simple enrollment in youth programs.

### Program Directors
Looking to modernize their camps, after-school programs, and payment systems.

## Critical Outcomes

### Trust
The design must look professional, secure, and established. Sloppy UI kills conversions.

### Speed
Parents on mobile devices need instant page loads.

### Clarity
Navigation and copy must clearly explain complex enrollment features (waitlists, approvals, payments).

## Mission

To persuade program directors that enrollment can be simple, transparent, and fair.

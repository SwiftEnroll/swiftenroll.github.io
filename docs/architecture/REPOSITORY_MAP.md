# Repository Map

## Directory Structure

| Directory | Responsibility |
|-----------|----------------|
| `content/` | **Primary workspace.** Markdown files for site pages. |
| `layouts/` | HTML templates. `_default` for page types, `partials` for components. |
| `assets/css/` | Tailwind source files (`main.css`). |
| `static/` | Raw assets (images, favicon). Content copied directly to `public/`. |
| `docs/` | Internal documentation. **Must stay updated with code.** |
| `hugo.toml` | **Single source of truth** for site config, menus, and global params. |
| `.lighthouserc.json` | Performance & Accessibility thresholds. |
| `public/` | Build output (git-ignored). Generated by `hugo --minify`. |
| `.github/` | GitHub Actions workflows (CI/CD, Lighthouse, link checks). |

## Configuration Files

### hugo.toml
- Site metadata (title, baseURL, description)
- Global parameters (`[params]`)
  - Company info (`[params.company]`)
  - Social links (`[params.social]`)
  - Form endpoints (`[params.forms]`)
  - CTA settings (`[params.cta]`)
- Menu structure (`[[menu.main]]`)
- Build settings

### tailwind.config.js
- Tailwind CSS configuration
- Custom color palette
- Typography settings
- Breakpoints
- Plugin configuration

### postcss.config.js
- PostCSS pipeline
- Tailwind CSS processing
- Autoprefixer configuration

### package.json
- npm scripts (start, build, watch)
- Dependencies (Tailwind, PostCSS, etc.)

### .lighthouserc.json
- Lighthouse CI configuration
- Performance thresholds
- URLs to test

## Build Artifacts (git-ignored)

- `public/` - Hugo build output
- `resources/` - Hugo resource cache
- `node_modules/` - npm dependencies
- `static/css/style.css` - Compiled Tailwind CSS (generated)
- `.hugo_build.lock` - Hugo build lock file

## Documentation Structure

## Critical Files

### Must Never Break
- `hugo.toml` - Site configuration
- `layouts/_default/baseof.html` - Base template
- `assets/css/main.css` - Tailwind entry point

### Hot Paths (Performance Critical)
- `content/_index.md` - Homepage
- `content/pricing.md` - Pricing page
- `layouts/index.html` - Homepage layout

### Integration Points
- Form endpoints in `hugo.toml` -> `[params.forms]`
- Analytics ID in `hugo.toml` -> `[params.googleAnalytics]`
- Turnstile site key in `hugo.toml` -> `[params.forms.turnstileSiteKey]`
